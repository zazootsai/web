如何在您的 Google 試算表啟用此後台？

步驟 1 請先在您的 Google 試算表（**工作表 1**）第 1 列建立以下 A 到 G 的欄位標題：
A1: 時間戳記 B1: name C1: title D1: phone E1: attendance F1: count G1: blessing
步驟 2 在試算表畫面，點擊上方選單的 擴充功能 (Extensions) > Apps Script，刪除原本所有程式碼，貼上為您此試算表量身打造的專屬程式碼： 
const sheetName = '工作表1' // 確保與您試算表左下角的名稱相同
const targetSpreadsheetId = '1LhtgTxnDwoO453997whqjKkvH-h9vJPv2Jx5SWygkUs' // 您的試算表專屬 ID

function doPost (e) {
  const lock = LockService.getScriptLock()
  lock.tryLock(10000)
  try {
    const doc = SpreadsheetApp.openById(targetSpreadsheetId)
    const sheet = doc.getSheetByName(sheetName)
    const headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0]
    const nextRow = sheet.getLastRow() + 1
    const newRow = headers.map(function(header) {
      return header === '時間戳記' ? new Date() : e.parameter[header]
    })
    sheet.getRange(nextRow, 1, 1, newRow.length).setValues([newRow])
    return ContentService
      .createTextOutput(JSON.stringify({ 'result': 'success', 'row': nextRow }))
      .setMimeType(ContentService.MimeType.JSON)
  } catch (e) {
    return ContentService
      .createTextOutput(JSON.stringify({ 'result': 'error', 'error': e.toString() }))
      .setMimeType(ContentService.MimeType.JSON)
  } finally {
    lock.releaseLock()
  }
}
步驟 3 在 Apps Script 專案視窗中：
  • 點擊上方 儲存檔案 (Save) 圖示。
  • 點擊右上角的 部署 (Deploy) 按鈕 > 選擇 新增部署 (New deployment)。
  • 點擊小齒輪圖示，選取類型為 網頁應用程式 (Web app)。
  • 執行身分： 選「我 (您的 Email)」
  • 誰有權限存取： 務必選擇「任何人 (Anyone)」
  • 點擊「部署」，過程中會彈出權限請求，請授權允許執行。
  • 部署完成後，複製畫面上產生的 **網頁應用程式 URL** (通常以 `/exec` 結尾)。
  • 將該網址貼回上方 **「串接您的 Google 試算表」** 欄位並儲存，就正式大功告成了！
